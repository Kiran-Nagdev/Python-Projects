# Python Project
📌 Project Overview

* This capstone project demonstrates practical use of Python, Pandas, and NumPy for data cleaning, transformation, merging, and analysis using multiple datasets. The project simulates an employee-project management system where project heads are assigned projects with different costs, statuses, and designation levels.

* The objective is to perform real-world data operations such as handling missing values, merging datasets, calculating bonuses, updating employee levels, and generating summary reports.

🎯 Objectives
* Create and manage multiple DataFrames using Python.
* Perform data cleaning and preprocessing.
* Handle missing values using running averages.
* Merge multiple datasets into one final dataset.
* Apply business rules for promotions, demotions, and bonuses.
* Generate project cost summaries.
* Filter records based on conditions.

### Technologies Used
* Python
* Pandas
* NumPy
* Jupyter Notebook
